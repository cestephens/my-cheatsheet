CSS Positioning and Display Reference


This page is designed to be a quick reference for many types of CSS properties and values that relate to positioning elements. The list is not complete and only contains some of the simpler properties/ values. To extend your knowledge there is some great information on flex and grid posistioning here.


Positioning Properties

The position type and offset properties are used together to define the location of and element on the page.

Position type

The position property specifies the position type for an element.

Value Description When you scroll? What's left where it was?
static Default value. Elements render in order, as they appear in the document flow Goes with the flow Err, it is where it is
relative Used in combo with offset properties to position the element relative to where it's own static position would have been Goes with the flow A gap
absolute Used in combo with offsets properties, the element is positioned relative to its first positioned (not static) ancestor element Goes with the flow No gap
fixed Used in combo with offset properties to posistion element relative to the viewport. Sticks around relative to the view port No gap
-webkit-sticky The object scrolls with the container until it reaches the offset defined with offset properties. Then it sticks. Goes with the flow, then sticks A gap

Offsets

I have grouped these properties as "offset" properties as they are used to define the offset from the default position based on an element's position type.

These properties define which edge you would like to offset from:

Property Description
left: Sets the left margin edge for a positioned element
right: Sets the right margin edge for a positioned element
top: Sets the top margin edge for a positioned element
bottom: Sets the bottom margin edge for a positioned element

The above offset properties can contain the following value types:

Value Description
auto Lets the browser calculate the edge position. This is default
absolute length Sets the edge position in px, cm, etc. Negative values are allowed
% length Sets the edge position in % of containing element. Negative values are allowed

Display Property

The display property determine the type of block the element takes when rendered.

Value Description
block Displays an element as a block element (like <p>). It starts on a new line, and takes up the whole width
inline Displays an element as an inline element (like <span>). Any height and width properties will have no effect
inline-block Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values
none Element is removed

Float Property

The float property determines where an element will sit in its container.

Value Description
right The element floats to the right of its container
left The element floats to the left of its container
right The default. The element does not float

Clear Property

The clear property specifies what should happen with the element that is next to a floating element.

Value Description
none Default. The element is not pushed below left or right floated elements
left The element is pushed below left floated elements
right The element is pushed below right floated elements
both The element is pushed below both left and right floated elements
